# 各位相加-简单

难度：简单<br />
<br />题目描述：<br />给定一个非负整数  `num`，反复将各个位上的数字相加，直到结果为一位数。<br />
<br />示例：<br />**输入:** `38`<br />**输出:** 2 <br />**解释: **各位相加的过程为**：**`3 + 8 = 11`, `1 + 1 = 2`。 由于 `2` 是一位数，所以返回 2。<br />
<br />解题思路：<br />对 9 取余<br />

```javascript
var addDigits = function (num) {
  if (num < 10) return num;
  return num % 9 || 9;
};
```
